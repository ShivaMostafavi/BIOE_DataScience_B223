
# Digital Biomarkers for Parkinson’s Disease Using Wearable Sensors and Questionnaires

This repository contains code for a research project investigating Parkinson’s disease classification using smartwatch sensor data and questionnaire information. Models evaluated include SVM, XGBoost, CNN, MOMENT-1, BioBERT-based questionnaire embeddings, and multimodal fusion approaches.

## Tasks
- PD vs Healthy Controls
- PD vs Other Movement Disorders
- Multi-class Classification

## Methods
- Signal feature extraction
- Questionnaire encoding with raw inputs and BioBERT
- Machine learning, deep learning, and fusion models


## Data
The project uses the PADS dataset. Data are not included in this repository.
